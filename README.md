# Primum

*First, do no harm.*

A clinical autopsy system for conversational AI in mental-health contexts. Primum stages adversarial multi-turn conversations between simulated patients and target language models, identifies clinical failures against DSM-grounded rubrics, and generates contrastive training pairs — turning every misstep into alignment data.

Built for the **Built with Opus 4.7** hackathon (Apr 21–26, 2026).

## Thesis

> *"There are billions of conversations happening right now between vulnerable people and language models. No one is reviewing them. Primum is the first system that does — and turns every failure into data for a safer model."*
